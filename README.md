# DNCL Language Support for VSCode (Experimental)

共通テスト手順記述標準言語 (DNCL) の記法を見やすくする Visual Studio Code 向け拡張機能です（非公式）。DNCLの実行環境は同梱していません。

シンタックスハイライティングは、拡張子`.dnc`でデフォルトで有効化されます。

## 機能

- シンタックスハイライティング
- よく使う記法の自動補完（Snippet）
    - 関数定義
    - If文 / If-Else文
    - For文 / While文 / Do-While文
